| 세그먼트        | 기능                                            |
| --------------- | ----------------------------------------------- |
| 코드 세그먼트   | 실행 가능한 프로그램의 기계어 코드를 포함       |
| 데이터 세그먼트 | 초기화된 전역 및 정적 변수 데이터를 포함        |
| BSS 세그먼트    | 초기화되지 않은 전역 및 정적 변수 데이터를 포함 |
| 스택 세그먼트   | 지역 변수 및 함수 호출 시 사용되는 데이터       |
| 힙 세그먼트     | 동적으로 할당된 메모리 데이터를 포함            |
